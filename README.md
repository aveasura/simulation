# Консольная симуляция экосистемы на клеточном поле.

- **R** (Rabbit, травоядное) ищет **G** (Grass)
- **F** (Fox, хищник) охотится на травоядных
- **T** (Tree) и **M** (Mountain) - препятствия
- `•` - пустая клетка

Проект построен вокруг пошагового цикла, рендера карты в консоль и управления во время выполнения (`pause/resume/quit`).

## Возможности

- Инициализация карты случайными сущностями (с гарантией, что есть `R`, `F`, `G`)
- Движение существ по кратчайшему пути (BFS, 4 направления)
- Взаимодействия:
    - травоядное съедает траву рядом с собой
    - хищник атакует травоядное рядом с собой
- Остановка симуляции, когда ни одно существо не может продвинуться к цели
- Управление из консоли:
    - `1` - пауза
    - `2` - продолжить
    - `3` - выход

## Требования

- JDK 21+
- Maven 3.9+

## Конфигурация

`Main` загружает `config.properties` из **classpath**.

Создай файл(если его нет) `resources/config.properties`:

```properties
width=10
height=10
quantity=20
delay=1000
```
## Параметры:

- width - ширина карты 
  - Значение по умолчанию: 5
  - Диапазон: 5..12

- height - высота карты
  - Значение по умолчанию: 5
  - Диапазон: 5..12

- quantity - количество сущностей на старте
  - Значение по умолчанию: 3
  - Диапазон: 3..(width*height)

- delay - задержка между ходами в миллисекундах
  - Значение по умолчанию: 1000
  - Диапазон: 500..5000

## Запуск
```bash
git clone https://github.com/aveasura/simulation.git
cd simulation
mvn clean compile
mvn exec:java -Dexec.mainClass=simulation.Main
```
